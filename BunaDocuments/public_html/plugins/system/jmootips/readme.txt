jmootips Package:

1. jmootips Joomla system plugin - With this plugin you will be able to create tooltips very easily
2. tipbutton Joomla editors-xtd plugin - Displays a button to make it possible to insert tooltips into an article.
   It displays a popup allowing you to choose the tooltip article
3. jmootips.css stylesheet for jmootips
4. javascripts for mootools and bootstrap framework
5. images for samples and ajax interface 
6. samples
